# Simulador-de-Sistemas---SO
Simulador de Sistemas Operacionais em C++ com interface gráfica GTK3. Implementa escalonamento de processos (Round-Robin, SJF Preemptivo, Prioridade Preemptiva) e gerência de memória virtual com paginação (FIFO, LRU, Ótimo). Projeto acadêmico — Engenharia de Computação 2026.
